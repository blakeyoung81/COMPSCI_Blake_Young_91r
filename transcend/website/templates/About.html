DNA analysis

Quantitative genetics is a field that attempts to account for phenotypic variation among individuals by using various methods of computational analysis. In the age of genome-wide association studies, there is an increasingly vast body of examinable information. The goal of this research project is to create a viable method whereby any two human genome SNP files can be synthesized, and the likelihood of a certain phenotypic expression in the offspring can be predicted. The project is an academic attempt at applying different, unconventional computational methods to a preexisting database of genome files. The project will involve creating a website, storing information on a database, coding several algorithms to analyze data, and real time rendering of several pages of results.
